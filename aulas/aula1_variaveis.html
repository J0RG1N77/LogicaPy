Aula 1: Variáveis - Armazenando Informações

No mundo real, usamos caixas, pastas ou gavetas para organizar e guardar objetos. Na programação, fazemos algo muito parecido com os dados. Para isso, usamos as variáveis!

O que são Variáveis?

Imagine uma variável como um pequeno espaço na memória do computador que possui um nome. Você pode colocar (ou "armazenar") informações dentro desse espaço, e o mais interessante é que o valor armazenado pode ser alterado a qualquer momento durante a execução do seu programa. Por isso o nome "variável"!

Pense assim:

  • Nome da caixa: É o nome da sua variável (ex: idade, nome_usuario, preco).
  • Conteúdo da caixa: É o valor que a variável está armazenando (ex: 25, "Maria", 19.99).

Quando você precisa usar essa informação em seu programa, basta "chamar" a variável pelo seu nome, e o computador saberá qual valor ele deve usar.

Por que precisamos de Variáveis?

Variáveis são fundamentais porque nos permitem:

  • Armazenar dados: Guardar informações que o programa precisa.
  • Reutilizar dados: Usar o mesmo valor em vários lugares do código sem ter que digitá-lo novamente.
  • Flexibilidade: Alterar o comportamento do programa com base nos dados que ele manipula.
  • Interagir com o usuário: Capturar entradas do usuário e armazená-las para processamento.

Declarando e Atribuindo Valores em Python

Uma das belezas do Python é a sua simplicidade. Para "criar" (declarar) uma variável e dar a ela um valor (atribuir), você só precisa usar o sinal de igual (=):

# Exemplo 1: Armazenando um número
idade = 30 

# Exemplo 2: Armazenando um texto (string)
nome = "Carlos Eduardo"

# Exemplo 3: Armazenando um número decimal (float)
altura = 1.82

# Exemplo 4: Uma variável pode mudar de valor
pontuacao = 0
print(pontuacao) # Saída: 0
pontuacao = 100 # Agora a variável pontuacao tem um novo valor
print(pontuacao) # Saída: 100

Note que você não precisa dizer ao Python que tipo de dado a variável vai guardar (número, texto, etc.). Ele é inteligente o suficiente para descobrir isso sozinho! Isso é chamado de tipagem dinâmica.

Regras para Nomes de Variáveis em Python

Escolher nomes para suas variáveis é importante para tornar seu código legível. Em Python, existem algumas regras simples:

  • Podem conter letras (a-z, A-Z), números (0-9) e o caractere underscore (_).
  • Devem começar com uma letra ou um underscore (NUNCA com um número).
  • São "case-sensitive" (sensíveis a maiúsculas e minúsculas). nome é diferente de Nome.
  • Não podem ser palavras-chave reservadas do Python (ex: if, for, while, print).

Boas práticas (convenções Python - PEP 8):

  • Use nomes descritivos (ex: idade_do_usuario é melhor que i).
  • Use letras minúsculas e separe palavras com underscores (snake_case): minha_primeira_variavel.
