Aula 4: Verdadeiro e Falso (Booleanos) e Negando Valores

Na aula de condicionais, falamos sobre como os programas tomam decisões baseadas em condições que são verdadeiras ou falsas. Agora, vamos entender mais a fundo esses valores e como manipulá-los.

Os Tipos de Dados Booleanos (bool)

Em programação, existe um tipo de dado específico para representar "verdadeiro" ou "falso": o tipo Booleano, ou bool em Python. Ele só pode ter dois valores:

  • True (Verdadeiro)
  • False (Falso)

Note que True e False sempre começam com letra maiúscula em Python. São valores especiais, não textos comuns.

# Exemplos de variáveis booleanas
chovendo = True
tem_desconto = False
usuario_logado = True

print(type(chovendo)) # Saída:

Negando Valores com o Operador not

Às vezes, queremos verificar o oposto de uma condição. Para isso, usamos o operador lógico not.

O not simplesmente inverte o valor booleano de uma expressão:

  • not True resulta em False
  • not False resulta em True

É como dizer "não é verdade que...".

# Exemplo com 'not'
esta_chovendo = False

if not esta_chovendo: # É o mesmo que if esta_chovendo == False:
    print("Ótimo dia para um passeio!")
else:
    print("Leve um guarda-chuva!")

# Outro exemplo:
tem_permissao = True
if not tem_permissao:
    print("Acesso negado!")
else:
    print("Bem-vindo!")

Usar not pode tornar suas condições mais legíveis em alguns casos.

Valores "Truthy" e "Falsy" em Python

Embora Python tenha os valores explícitos True e False, muitos outros tipos de dados podem ser avaliados como verdadeiros ou falsos em um contexto booleano (como em uma condição if).

Esses são conhecidos como valores "truthy" (se comportam como True) e "falsy" (se comportam como False).

Os principais valores falsy em Python são:

  • False (o próprio valor booleano)
  • None (o valor nulo)
  • 0 (o número zero, tanto inteiro quanto float: 0, 0.0)
  • "" (string vazia)
  • [] (lista vazia)
  • () (tupla vazia)
  • {} (dicionário vazio)
  • set() (conjunto vazio)

Todos os outros valores, na maioria dos casos, são considerados truthy.

# Exemplos de valores truthy/falsy
if 0:
    print("Este não será impresso.")
else:
    print("0 é falsy.") # Saída: 0 é falsy.

if "Olá":
    print("String não vazia é truthy.") # Saída: String não vazia é truthy.

if []:
    print("Este não será impresso.")
else:
    print("Lista vazia é falsy.") # Saída: Lista vazia é falsy.

Entender "truthy" e "falsy" é importante para escrever condições mais concisas e pythonicas.
