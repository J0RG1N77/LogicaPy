Aula 2: Tipos de Dados, Expressões e Números

Na aula anterior, aprendemos que variáveis são como "caixas" para guardar informações. Mas que tipo de informações podemos guardar? É aí que entram os Tipos de Dados!

O que são Tipos de Dados?

Assim como no mundo real temos diferentes tipos de objetos (números, palavras, imagens, sons), na programação, as informações também são classificadas em tipos. Um Tipo de Dado define a natureza do valor que uma variável pode armazenar e as operações que podem ser realizadas com ele.

Em Python, por ser uma linguagem de tipagem dinâmica, você não precisa declarar o tipo explicitamente, mas é fundamental entender que o tipo existe e afeta como o Python trata a informação. Os tipos de dados mais comuns que veremos agora são os numéricos e os de texto (string).

Números em Python

Python lida muito bem com números e oferece dois tipos numéricos principais para começar:

  • Inteiros (int): São números positivos ou negativos sem casas decimais. 
    idade = 30
quantidade = -5
ano = 2024
  • Ponto Flutuante (float): São números que possuem casas decimais. Eles são usados para representar valores mais precisos. 
    preco = 19.99
altura = 1.75
temperatura = -3.5

Python consegue automaticamente identificar se um número é um int ou um float pela presença (ou ausência) do ponto decimal.

Expressões Numéricas: Operadores Aritméticos

Agora que sabemos guardar números, como fazemos cálculos com eles? Usamos operadores aritméticos para criar expressões. Uma expressão é uma combinação de valores (números, variáveis) e operadores que o Python pode "calcular" para produzir um novo valor.

Os operadores aritméticos mais comuns em Python são:

  • + : Adição (soma)
  • - : Subtração
  • * : Multiplicação
  • / : Divisão (sempre retorna um float)
  • % : Módulo (retorna o resto da divisão)
  • ** : Exponenciação (potência)
  • // : Divisão Inteira (retorna a parte inteira da divisão)
# Exemplos de expressões:
resultado_soma = 10 + 5    # resultado_soma será 15
resultado_sub = 20 - 7     # resultado_sub será 13
resultado_mult = 4 * 6     # resultado_mult será 24
resultado_div = 10 / 3     # resultado_div será 3.333... (float)
resto_div = 10 % 3       # resto_div será 1 (10 dividido por 3 é 3 com resto 1)
potencia = 2 ** 3        # potencia será 8 (2 elevado a 3)
div_inteira = 10 // 3    # div_inteira será 3 (parte inteira da divisão)

# Você também pode usar variáveis em expressões:
preco_unitario = 5.50
quantidade = 3
total = preco_unitario * quantidade # total será 16.50

A ordem de precedência dos operadores é a mesma da matemática (parênteses primeiro, depois potências, multiplicação/divisão, e por fim adição/subtração).

Strings (Textos) em Python

Além de números, a maioria dos programas lida com texto. Em programação, sequências de caracteres (letras, números, símbolos) são chamadas de strings (do inglês, "cadeias de caracteres").

Em Python, você define uma string colocando o texto entre aspas simples (' ') ou aspas duplas (" ").

  • str: Representa texto. 
    nome = "Alice"
mensagem = 'Bem-vindo ao Python!'
letra = "A" # Até um único caractere é uma string

Você pode usar aspas simples ou duplas, o importante é começar e terminar com o mesmo tipo de aspas.

Concatenação de Strings

Um operador muito comum com strings é o +, que serve para "juntar" ou "concatenar" strings. Diferente dos números, onde o + soma, com strings ele une o texto.

saudacao = "Olá, "
nome_aluno = "Maria"
boas_vindas = saudacao + nome_aluno + "!" # boas_vindas será "Olá, Maria!"

print(boas_vindas)

É importante lembrar que você só pode concatenar strings com outras strings. Tentar somar um número diretamente com uma string causará um erro em Python, a menos que você converta o número para string primeiro.

primeiro_nome = "Ana"
sobrenome = "Silva"
nome_completo = primeiro_nome + " " + sobrenome
print(nome_completo)?
