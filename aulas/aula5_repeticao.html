Aula 5: Laços de Repetição (Loops) - Repetindo Tarefas

Imagine que você precisa imprimir a mesma mensagem 100 vezes, ou processar cada item em uma longa lista. Digitar o código 100 vezes seria exaustivo e ineficiente, certo? É para isso que existem os Laços de Repetição (ou "Loops")!

O que são Laços de Repetição?

Laços de repetição são estruturas de controle que permitem executar um bloco de código repetidamente, várias vezes, sem ter que reescrevê-lo. Eles são a base da automação em programação.

Em Python, os dois tipos principais de laços que vamos aprender são o for e o while.

O Laço for (Para Cada)

O laço for é usado para iterar (percorrer) sequências de itens, como listas, strings, ou ranges de números. Ele executa um bloco de código uma vez para cada item da sequência.

A função range() é frequentemente usada com o for para gerar sequências de números:

  • range(5): Gera números de 0 a 4.
  • range(1, 5): Gera números de 1 a 4.
  • range(0, 10, 2): Gera números de 0 a 9, pulando de 2 em 2 (0, 2, 4, 6, 8).
# Exemplo 1: Imprimindo números de 0 a 4
print("Contagem regressiva com for:")
for i in range(5): # O 'i' é a variável que recebe cada valor da sequência
    print(i)
# Saída:
# 0
# 1
# 2
# 3
# 4

# Exemplo 2: Percorrendo uma lista de frutas
frutas = ["maçã", "banana", "cereja"]
for fruta in frutas:
    print(f"Eu gosto de {fruta}.")
# Saída:
# Eu gosto de maçã.
# Eu gosto de banana.
# Eu gosto de cereja.

Assim como nas condicionais, o bloco de código dentro do for deve ser indentado.

Qual será a saída do seguinte código Python?
for numero in range(1, 4):
    print(numero)

Qual será a saída do seguinte código Python?
linguagens = ["Python", "Java", "C#"]
for lang in linguagens:
    print(lang + " é legal!")

O Laço while (Enquanto)

O laço while é usado para executar um bloco de código enquanto uma condição for verdadeira. Ele continua repetindo até que a condição se torne falsa.

É crucial que a condição do while eventualmente se torne falsa, caso contrário, você terá um loop infinito, e seu programa nunca terminará!

# Exemplo: Contador simples
contador = 0

while contador < 5:
    print(contador)
    contador = contador + 1 # Importante: Atualizar a condição para evitar loop infinito
# Saída:
# 0
# 1
# 2
# 3
# 4

Sempre garanta que haja uma forma da condição do while se tornar falsa para o laço parar.

Qual será a saída do seguinte código Python?
x = 0
while x < 3:
    print("Olá")
    x += 1

break e continue em Laços

Às vezes, precisamos de um controle mais refinado sobre os laços. É para isso que existem break e continue:

  • break: Interrompe completamente o laço e sai dele. O programa continua a execução da primeira linha após o laço.
  • continue: Pula a iteração atual do laço e vai para a próxima iteração. O restante do código no bloco atual do laço é ignorado.
# Exemplo com 'break'
for numero in range(10):
    if numero == 5:
        break # Interrompe o laço quando numero for 5
    print(numero)
# Saída: 0, 1, 2, 3, 4 (o 5 não é impresso porque o laço quebra antes)

# Exemplo com 'continue'
for i in range(5):
    if i == 2:
        continue # Pula esta iteração (o 2 não será impresso)
    print(i)
# Saída: 0, 1, 3, 4

Qual será a saída do seguinte código Python?
total = 0
for i in range(1, 6):
    if i == 3:
        break
    total += i
print(total)

Qual será a saída do seguinte código Python?
palavra = "Python"
for letra in palavra:
    if letra == "t":
        continue
    print(letra)
