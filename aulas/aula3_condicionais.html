Aula 3: Condicionais - Fazendo o Programa Tomar Decisões

Até agora, nossos programas seguem um caminho direto, linha após linha. Mas o que acontece se quisermos que o programa faça algo apenas se uma certa condição for verdadeira? É aí que entram as condicionais!

O que são Condicionais?

Condicionais são estruturas que permitem ao seu programa tomar decisões. Elas verificam se uma determinada condição é verdadeira ou falsa e, com base nisso, executam um bloco de código específico ou o ignoram. Pense nelas como a lógica do "SE... ENTÃO..." (ou "SE... ENTÃO... SENÃO...").

Em Python, as condicionais são construídas principalmente com as palavras-chave if, elif (else if) e else.

O Bloco if (Se)

O bloco if é a estrutura condicional mais básica. Ele executa um código APENAS se a condição especificada for verdadeira.

# Exemplo: Verificando se a pessoa é maior de idade
idade = 20

if idade >= 18:
    print("Você é maior de idade e pode dirigir.")
    print("Bem-vindo(a) ao clube dos adultos!") # Este também será executado se a condição for verdadeira

Pontos importantes:

  • A condição (ex: idade >= 18) é uma expressão booleana que resulta em True ou False.
  • A condição termina com dois pontos (:).
  • O código que faz parte do if deve ter uma indentação (geralmente 4 espaços ou 1 tab). Essa indentação é como o Python sabe o que pertence ao bloco if.

Qual será a saída do seguinte código Python?
temperatura = 28
if temperatura > 25:
    print("Está quente!")

O Bloco else (Senão)

O bloco else é usado em conjunto com o if para fornecer uma alternativa. Ele é executado SOMENTE se a condição do if for falsa.

# Exemplo: Aprovação em um teste
nota = 7.5

if nota >= 7.0:
    print("Parabéns! Você foi aprovado(a).")
else:
    print("Você precisa estudar mais para a próxima vez.")

O else também termina com dois pontos (:) e o código dentro dele deve ser indentado.

Qual será a saída do seguinte código Python?
idade = 16
if idade >= 18:
    print("Adulto")
else:
    print("Menor")

O Bloco elif (Senão Se / Else If)

Quando você tem múltiplas condições a serem verificadas em sequência, o elif é a sua ferramenta. Ele é usado entre um if e um else (opcional) para testar condições adicionais.

O Python verifica as condições em ordem. Assim que uma condição (if ou elif) é verdadeira, o bloco de código correspondente é executado e o restante da estrutura condicional é ignorado.

# Exemplo: Classificação de notas
pontuacao_final = 85

if pontuacao_final >= 90:
    print("Nota: A")
elif pontuacao_final >= 80: # Se o primeiro 'if' foi falso, testa esta condição
    print("Nota: B")
elif pontuacao_final >= 70: # Se o 'elif' anterior foi falso, testa esta
    print("Nota: C")
else: # Se nenhuma das anteriores foi verdadeira
    print("Nota: D ou F")

Um programa deve classificar um número: positivo, negativo ou zero. Se numero = -10, qual será a saída do código?
numero = -10
if numero > 0:
    print("Positivo")
elif numero < 0:
    print("Negativo")
else:
    print("Zero")

Operadores de Comparação (para Condições)

Para construir as condições nos blocos if, elif e else, usamos operadores de comparação. Eles sempre retornam um valor booleano (True ou False).

  • == : Igual a (atenção: um único = é para atribuição!)
  • != : Diferente de
  • > : Maior que
  • < : Menor que
  • >= : Maior ou igual a
  • <= : Menor ou igual a
x = 10
y = 5

print(x == y)  # Saída: False
print(x != y)  # Saída: True
print(x > y)   # Saída: True
print(x <= y)  # Saída: False

Operadores Lógicos (Combinando Condições)

Às vezes, você precisa verificar mais de uma coisa ao mesmo tempo. Para isso, usamos operadores lógicos:

  • and : Retorna True se AMBAS as condições são verdadeiras.
  • or : Retorna True se PELO MENOS UMA das condições é verdadeira.
  • not : Inverte o resultado de uma condição (True vira False e vice-versa).
# Exemplo com 'and'
idade = 25
tem_carteira = True

if idade >= 18 and tem_carteira == True:
    print("Pode dirigir!")

# Exemplo com 'or'
dia = "Domingo"
humor = "bom"

if dia == "Sábado" or dia == "Domingo" or humor == "bom":
    print("É um ótimo dia para relaxar!")

# Exemplo com 'not'
esta_chovendo = False

if not esta_chovendo:
    print("Vamos sair!")

Qual será a saída do seguinte código Python?
saldo = 500
compra = 600
cartao_disponivel = True

if saldo >= compra or cartao_disponivel:
    print("Compra pode ser realizada.")
else:
    print("Compra negada.")
