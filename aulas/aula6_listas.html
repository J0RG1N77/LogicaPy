Aula 6: Listas e Tuplas - Organizando Coleções de Dados

Até agora, aprendemos a guardar valores em variáveis individuais. Mas o que acontece se quisermos guardar vários valores relacionados em uma única "caixa"? É para isso que usamos as Listas e Tuplas em Python!

O que são Listas?

Uma lista em Python é uma coleção ordenada de itens. Pense nela como uma lista de compras, onde você pode ter vários itens e a ordem deles importa. As listas são mutáveis, o que significa que você pode adicionar, remover ou modificar seus elementos após a criação.

As listas são definidas usando colchetes [] e os itens são separados por vírgulas.

# Exemplo: Criando uma lista de frutas
frutas = ["maçã", "banana", "cereja", "laranja"]
numeros = [1, 5, 8, 12, 20]
mistura = ["texto", 10, True, 3.14] # Listas podem ter tipos de dados diferentes

Acessando Elementos da Lista (Indexação)

Cada item em uma lista tem uma posição, chamada de índice. Em Python (e na maioria das linguagens de programação), a contagem dos índices começa em 0.

  • Primeiro item: índice 0
  • Segundo item: índice 1
  • E assim por diante...

Você pode acessar um elemento específico da lista usando seu índice entre colchetes.

minha_lista = ["Python", "Java", "C++", "JavaScript"]

print(minha_lista[0]) # Saída: Python (o primeiro elemento)
print(minha_lista[2]) # Saída: C++ (o terceiro elemento)

# Você também pode usar índices negativos para contar do final da lista
print(minha_lista[-1]) # Saída: JavaScript (o último elemento)
print(minha_lista[-2]) # Saída: C++ (o penúltimo elemento)

Qual será a saída do código Python?
cores = ["vermelho", "azul", "verde", "amarelo"]
print(cores[1])

Modificando Listas (Mutabilidade)

Como as listas são mutáveis, você pode facilmente mudar o valor de um elemento, adicionar novos ou remover existentes.

  • Alterar valor: Acesse o elemento pelo índice e atribua um novo valor.
  • Adicionar: Use o método .append() para adicionar no final, ou .insert() para adicionar em uma posição específica.
  • Remover: Use .remove() para remover um valor específico, ou .pop() para remover pelo índice.
minha_lista = ["carro", "moto", "bicicleta"]

# Alterar um elemento
minha_lista[0] = "avião"
print(minha_lista) # Saída: ['avião', 'moto', 'bicicleta']

# Adicionar um elemento no final
minha_lista.append("ônibus")
print(minha_lista) # Saída: ['avião', 'moto', 'bicicleta', 'ônibus']

# Remover um elemento pelo valor
minha_lista.remove("moto")
print(minha_lista) # Saída: ['avião', 'bicicleta', 'ônibus']

# Remover um elemento pelo índice (e pode retornar o elemento removido)
elemento_removido = minha_lista.pop(0) # Remove o 'avião' (índice 0)
print(minha_lista) # Saída: ['bicicleta', 'ônibus']
print(f"Elemento removido: {elemento_removido}") # Saída: Elemento removido: avião

Qual será o conteúdo da lista `numeros` após a execução deste código?
numeros = [10, 20, 30]
numeros.append(40)
numeros[0] = 5

O que são Tuplas?

Uma tupla em Python também é uma coleção ordenada de itens, muito parecida com uma lista. A grande diferença é que as tuplas são imutáveis. Isso significa que, uma vez criada, você NÃO pode adicionar, remover ou modificar seus elementos.

As tuplas são definidas usando parênteses () e os itens são separados por vírgulas.

# Exemplo: Criando uma tupla de coordenadas
coordenadas = (10.0, 20.5)
dias_da_semana = ("Segunda", "Terça", "Quarta", "Quinta", "Sexta", "Sábado", "Domingo")

# Tentativa de modificar (irá gerar um erro!)
# coordenadas[0] = 15.0 # Isso causaria um TypeError

Tuplas são ideais para coleções de dados que não devem mudar, como coordenadas geográficas, configurações fixas ou quando você quer garantir que os dados permaneçam intactos.

Acessando Elementos da Tupla (Indexação)

Assim como nas listas, você acessa os elementos de uma tupla por seu índice, que também começa em 0.

minha_tupla = ("João", 25, "Engenheiro")

print(minha_tupla[0]) # Saída: João
print(minha_tupla[1]) # Saída: 25
